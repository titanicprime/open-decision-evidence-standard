# Security Policy

This repository welcomes reports about vulnerabilities in supporting materials and security concerns in the draft specification itself.

In scope are concerns such as:

- cryptographic misuse
- signature confusion
- replay risk
- revocation failure
- stale record acceptance
- selective disclosure leakage
- issuer impersonation
- verifier endpoint abuse
- privacy leakage
- malicious conformance claims

Report concerns to **security@example.com**.

**Placeholder notice:** `security@example.com` is a placeholder address and should be replaced before broad release.

Please include a concise description of the issue, affected files or record elements, potential impact, and any suggested mitigation.
