# Agentic AI Profile

This profile sketches additional expectations for agentic AI or delegated automation contexts.

Expected additions or stricter practices include:

- clear agent or system identity;
- explicit delegated authority basis;
- declared tool access boundaries;
- a runtime profile capturing execution context;
- action boundary descriptions for what the agent may and may not do;
- explicit human intervention state where applicable;
- policy constraints governing the agent path;
- freshness and revocation handling suitable for delegated execution; and
- evidence commitments that preserve logs or execution traces without requiring full disclosure.
