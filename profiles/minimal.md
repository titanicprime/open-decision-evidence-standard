# pder_v0_1_minimal

`pder_v0_1_minimal` is the baseline profile for the discussion draft.

It expects a record that:

- validates against the base schema;
- identifies the decision, issuer, and decision timestamp;
- includes authority coordinates;
- distinguishes human and machine participation;
- includes an evidence commitment or an explicit `none` value;
- includes consumption conditions;
- includes freshness information; and
- includes verification metadata or an explicit `none` placeholder.
